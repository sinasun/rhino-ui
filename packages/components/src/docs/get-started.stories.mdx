# Get Started

### 1. Install

<Source language='bash' code={`
yarn add @rhino-ui/components
`}/>

or

<Source language='bash' code={`
npm install @rhino-ui/components
`}/>

### 2. Import Global CSS

<Source code={`
@import '@rhino-ui/components/dist/css/utilities.css';
@import '@rhino-ui/components/dist/css/variables.css';
@import '@rhino-ui/components/dist/css/index.css';
`}
/>

### 3. Import Global CSS Reset (Optional)

We recommend importing our global reset in order to maintain a consistent
look of all components across applications.

<Source code={`
@import '@rhino-ui/components/dist/css/reset.css'
`}/>

### 4. Import Nexa Font (Optional)

We recommend importing our brand font, Nexa, for styling headings, etc.

<Source code={`
@import '@rhino-ui/components/dist/css/fonts';
`} />

### 5. Basic Usage

<Source code={`
import React from 'react';
import ReactDOM from 'react-dom';
import { Button } from '@rhino-ui/components';

function App() {
  return <Button>Hello World</Button>;
}

ReactDOM.render(<App />, document.querySelector('#app'));
`}/>
